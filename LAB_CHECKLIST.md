# Udacity Azure ML Project Checklist

Use this checklist during your lab session to avoid losing work.

## Before Starting

- Confirm repository remote is set
- Pull latest changes
- Install dependencies
- Verify Azure authentication
- Verify workspace config

## Stage Checklist

1. Run Automated ML experiment
- Capture screenshot of completed run
- Save or copy best model details
- Commit and push

2. Deploy best model
- Capture endpoint healthy status screenshot
- Save scoring URI and model version
- Commit and push

3. Enable logging
- Export logs to artifacts/service_logs.txt
- Capture screenshot of logs or App Insights confirmation
- Commit and push

4. Export Swagger
- Save artifacts/swagger.json
- Capture swagger UI or endpoint docs screenshot
- Commit and push

5. Consume endpoint
- Run sample scoring request
- Capture terminal output screenshot
- Commit and push

6. Create and publish pipeline
- Capture successful pipeline run screenshot
- Capture published pipeline endpoint screenshot
- Commit and push

7. Documentation and screencast
- Fill README architecture and key steps
- Add all screenshot links
- Record screencast walkthrough
- Commit and push final submission

## Lab Time Safety

- Push to GitHub after each stage
- Do not wait until end of session
- Reserve time for final README cleanup and screencast upload
